# git-ignore

Initialize a `.gitignore` file based on templates from GitHub.

## Install

`npm install -g git-ignore`

## Usage

- `ignore node`: Generates a `.gitignore` file for a NodeJS project.
- `ignore`: Prompts you to select from a list of languages.
